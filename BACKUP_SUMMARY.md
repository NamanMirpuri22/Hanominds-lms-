## 🔐 How to Restore/Deploy

### From This Backup

1. **Copy Entire hanolms Folder**
   ```bash
   cp -r hanolms /new/location/
   ```

2. **Install Dependencies**
   ```bash
   cd hanolms
   python -m venv .venv
   .venv\Scripts\Activate.ps1  # Windows
   pip install -r requirements.txt
   ```

3. **Run Application**
   ```bash
   python app.py
   ```

4. **Access at**: http://127.0.0.1:5000
